# Changelog

All notable changes to this project will be documented in this file.
Alle wichtigen Änderungen an diesem Projekt werden in dieser Datei festgehalten.

## [0.3.1] - 2026-05-14

### English
#### Fixed
- **Mobile UI**: Corrected vertical alignment and grouping of "Unit Price" in the order history.
- **Visual Consistency**: Unified price and profit coloring across desktop and mobile views.

---

### Deutsch
#### Behoben
- **Mobile UI**: Korrektur der vertikalen Ausrichtung und Gruppierung des Stückpreises in der Auftragshistorie.
- **Visuelle Konsistenz**: Vereinheitlichung der Farbgebung für Preise und Profit über Desktop- und Mobile-Ansichten hinweg.

## [0.3.0] - 2026-05-14

### English
#### Added
- **Batch Printing Mode**: Calculate multiple identical objects in a single job.
- **Dynamic Unit Price**: Auto-calculates per-piece costs with reorder warnings.
- **Expanded Localization**: All batch features translated into 22 languages.
- **UX Improvement**: Input fields auto-select text on focus for easier overwriting.

#### Changed
- **Layout Optimization**: Toggle switches for batch and supports are now aligned.
- **Order History**: Batch info and unit prices clearly displayed in the list view.

#### Fixed
- **Validation**: Saving orders with supports now requires post-processing time > 0.
- **Layout Fix**: Corrected desktop display issues for support toggle.

---

### Deutsch
#### Hinzugefügt
- **Batch-Druck Modus**: Ermöglicht die Kalkulation mehrerer identischer Objekte in einem Druckauftrag.
- **Dynamische Stückpreis-Anzeige**: Automatische Berechnung des Einzelpreises inkl. Warnhinweis für Nachbestellungen.
- **Erweiterte Lokalisierung**: Alle Batch-Funktionen in 22 Sprachen übersetzt.
- **Eingabe-Komfort**: Automatisches Markieren von Zahlenfeldern beim Fokussieren für schnelleres Überschreiben von Standardwerten.

#### Geändert
- **Layout-Optimierung**: Toggle-Schalter für Batch und Stützstrukturen sind nun konsistent untereinander angeordnet.
- **Tabellen-Ansicht**: Batch-Informationen und Stückpreise werden nun übersichtlich in der Auftragshistorie angezeigt.

#### Behoben
- **Validierung**: Das Speichern von Aufträgen mit Stützstrukturen erfordert nun zwingend eine Nachbearbeitungszeit > 0.
- **Layout-Fix**: Korrektur von Verschiebungen bei der Anzeige von Stützstrukturen auf Desktop-Geräten.
